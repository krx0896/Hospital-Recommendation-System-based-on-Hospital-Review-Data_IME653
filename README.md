# 1. Introduction
본 과제는 네이버의 한글 병원 리뷰 데이터를 활용하여 사용자가 프롬프트로 입력한 조건과 가장 잘 어울리는 병원을 추천해주기 위해 텍스트 비정형 데이터에 대해 전처리, 토픽 모델링, 후처리, 워드 임베딩, 감성 분석, 유사도 계산, 유사도 가중치 설계 및 적용을 하여 추천 시스템을 설계하였다.
